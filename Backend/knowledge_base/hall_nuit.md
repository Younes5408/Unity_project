# Hall de Nuit (Night Hall) - Guide Architectural

## Dimensions Recommandées

- **Minimum:** 3-4 m² (2m x 1.5m)
- **Standard:** 4-6 m²
- **Grand:** 6-8 m²

## Fonction Principale

Hall ou dégagement desservant:
- Chambres
- Salle de bain
- Eventuellement WC séparé
- Circulation sans passer par pièces communes

## Caractéristiques Essentielles

### Structure
- Espace central de circulation
- Portes des chambres donnant dessus
- Accès salle de bain
- Pas de "cul-de-sac" (éviter impasses)

### Dimensions Couloir
- Largeur minimum: 1.2m (confortable: 1.5m)
- Longueur variable selon nombre chambres

## Éclairage

### Lumière Naturelle
- Rarement fenêtre directe
- Lumière indirecte depuis chambres acceptable

### Lumière Artificielle
- Éclairage obligatoire (plafond ou appliques)
- Interrupteur à l'entrée
- Éclairage de nuit recommandé (veilleuse LED)

## Circulation et Agencement

### Portes Aboutissant sur Hall
- Maximum 4-5 portes pour hall 4m²
- Espace dégagement: 0.9m minimum devant chaque porte
- Portes ne doivent pas se croiser (danger)

### Rangements
- Placard/armoire possible le long mur
- Profondeur: 0.5m-0.6m
- Ne pas bloquer circulation

## Configuration Multi-Chambres

### Petit Logement (2 chambres)
```
[Chambre 1]
[Hall 4m²] [Salle de bain]
[Chambre 2]
```

### Logement Standard (3 chambres)
```
     [Chambre 1]
[Chambre 2]
[Hall 5-6m²] [Salle de bain + WC]
[Chambre 3]
```

### Grand Logement (4+ chambres)
```
[Ch1] [Salle de bain]
      [Hall 6-8m²]
[Ch2]   [WC]
[Ch3] [Chambre parent]
```

## Proximité et Localisation

### Connexion au Reste Logement
- Accès depuis entrée ou couloir principal
- Peut être porte fermée (isolation)
- Crée zone privée séparée du salon/cuisine

### Avantages
- Intimité des chambres protégée
- Pas de passage par salon pour chambre
- Isolation sonore/thermique améliorée
- Salle de bain pour usagers chambre

## Aspect Technique

### Portes
- Largeur standard: 0.85m
- Portes avec ventilation (grille) pour circulation air
- Fermeture recommandée (intimité)

### Ventilation
- Aération par salle de bain ou WC
- Pas d'équipement humide dans hall
- Évaporation depuis salle de bain OK

## Optionnel - Aménagements Additionnels

- Petit coin lecture (si espace > 6m²)
- Rangement linéaire (linge chambre)
- Placard aspirateur/nettoyage
- Vestiaire léger (cintres, étagères)

## Attention - Erreurs à Éviter

- Ne pas faire hall trop étroit (< 1.2m)
- Ne pas créer trop de portes (circulation chaotique)
- Ne pas bloquer passage avec meubles
- Ne pas oublier éclairage
- Ne pas créer impasses (cul-de-sac)
