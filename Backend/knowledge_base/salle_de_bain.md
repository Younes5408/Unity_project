# Salle de Bain - Guide Architectural

## Dimensions Recommandées

- **Salle de bain minimum:** 4-5 m² (2m x 2.2m)
- **Salle de bain standard:** 6-8 m²
- **Salle de bain spacieuse:** 8-12 m²

## Équipements Minimaux

### Obligatoires
- Baignoire OU douche (au moins l'un des deux)
- Toilettes (WC)
- Lavabo/vasque
- Miroir

### Recommandés
- Baignoire ET douche (pour grand logement)
- Meubles de rangement
- Chauffage (radiateur ou chauffage au sol)

## Aération - CRITIQUE

- **OBLIGATION:** Extraction d'air mécanique (VMC)
- Ou fenêtre d'aération minimum 0.5m x 0.5m
- **Recommandé:** Fenêtre + VMC pour meilleur confort
- Fenêtre dégradée ou givrée acceptable

## Humidité et Ventilation

- VMC obligatoire: élimine humidité, moisissures
- Capacité: minimum 30 m³/h standard
- 50 m³/h recommandé pour cabine douche
- Gaines d'évacuation vers extérieur obligatoire

## Accès et Proximité

### Localisation Recommandée
- Proche des chambres (distances courtes)
- Proche cuisine si possible
- Salle de bain parent: dans/près chambre parents
- Salle de bain enfants: accès hall/dégagement

### Porte et Passage
- Largeur porte: minimum 0.80m (0.85m recommandé)
- Espace entrée: 0.6m minimum
- Porte vers chambre d'eau conseillée (intimité)

## Configuration Intérieure

### Salle de Bain 4-5 m²
```
Disposition typique:
[Toilettes] | [Lavabo]
             [Baignoire/Douche]
```

### Salle de Bain 6-8 m²
```
[Baignoire] [Douche]
[Toilettes] [Lavabo + Rangements]
```

### Salle de Bain 8-12 m²
```
[Baignoire spacieuse]
[Douche séparée] [Toilettes]
[Lavabo double + Rangements]
```

## Revêtements et Matériaux

- Carrelage obligatoire ou revêtement hydrofuge
- Joints étanches
- Peinture anti-humidité (si peinture)

## Équipement Électrique

- Prise rasoir (si obligatoire)
- Éclairage principal + miroir lumineux
- Interrupteur éloigné douche/baignoire (sécurité)

## Chauffage

- Radiateur soufflant ou chauffage au sol
- Température de confort: 20-24°C
- Sèche-serviettes recommandé (confort + fonctionnel)

## Accessibilité

- Douche praticable: minimum 0.8m x 0.8m
- Barre d'appui recommandée
- Seuil bas ou accessible en fauteuil roulant (si nécessaire)
