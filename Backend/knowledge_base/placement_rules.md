# Règles de Placement Architectural - Guide Complet

## Hierarchie des Zones

### Zone Publique / Commune
- Salon
- Cuisine
- Salle à manger
- Entrée
- **Caractéristique:** Accès libre, accueil visiteurs

### Zone Circulation / Transition
- Couloir principal
- Hall d'entrée
- Hall de nuit
- Dégagements
- **Caractéristique:** Passage, liaison zones

### Zone Privée / Intime
- Chambres
- Salle de bain
- WC privés
- **Caractéristique:** Intimité, fermeture, accès limité

## Règles de Proximité Recommandée

### Proximité à Privilégier

**Cuisine ↔ Salle à Manger**
- Très proches (idéalement jointure directe)
- Service facile des repas
- Minimise trajets

**Cuisine ↔ Entrée/Hall**
- Proximité recommandée (pas plus de 2 pièces)
- Facilite livraisons, marché
- Odeurs vers extérieur meilleures

**Salon ↔ Cuisine**
- Ouverture possible (trend moderne)
- Ou séparation simple (classique)
- Dépend préférences utilisateur

**Chambre ↔ Salle de Bain**
- Très proche (idéalement 1 pièce distance)
- Accès rapide sans traverser logement
- Confort maximum

**Chambre Enfant ↔ Salle de Bain Commune**
- Hall nuit optimale pour accès
- Sans passer par zone adulte
- Chambres enfants proches les unes des autres

### Éloignement Obligatoire

**Cuisine ↔ Chambre LOIN**
- Distance minimum: 1-2 pièces d'intermédiaire
- Raison: odeurs, bruit, vapeur humide
- Isolation acoustique/thermale

**Salon ↔ Chambre ÉLOIGNÉS**
- Distance minimum: 1 pièce d'intermédiaire
- Raison: bruit, lumière, visites
- Surtout chambres enfant

**Salle de Bain ↔ Salon LOIN**
- Distance minimum: 1 pièce
- Raison: bruit eau, humidité, intimité
- Visuellement invisible

## Principes de Circulation

### Circulation Logique
- Pas de "dead-end" (cul-de-sac)
- Flux naturel depuis entrée
- Pas de traversée chambre pour accès autre chambre
- Pas de passage zone privée vers zone publique inévitale

### Largeurs de Passage
- Couloir principal: 1.5m minimum (confortable: 1.8m)
- Couloir secondaire: 1.2m minimum
- Dégagement portes: 0.9m minimum chaque côté

## Utilisation Optimale de l'Espace

### Pièces d'Angle
- Avantage: 2 fenêtres possibles
- Idéal pour: Salon, grande chambre, cuisine
- À éviter pour: WC, petit dégagement

### Pièces Profondes
- Longueur > 2x largeur = problématique
- Solution: division visuelle, partition partielle
- Lighting important pour pièces profondes

### Pièces Étroites
- Largeur < 3m: limiter longueur (max 5-6m)
- Aménagement difficile (meubles encombrants)
- Circulation réservée passages

## Lumière et Fenêtres

### Distribution Fenêtres
- **Pièces communes (salon, cuisine):** Priorité fenêtres
- **Chambres:** Fenêtre obligatoire par code
- **Salle de bain:** Fenêtre + VMC optimal
- **Hall, couloir:** Rarement fenêtre

### Orientations Idéales
- **Nord:** Cuisine (lumière stable, pas surchauffe)
- **Sud/Ouest:** Salon (lumière abondante, ambiance)
- **Est:** Chambre parent (lumière matin douce)
- **Ouest:** Chambre enfants (moins sun-glare)

## Flux Humain et Usage

### Matin (Départ)
```
Chambre → Salle de bain → Cuisine → Entrée (rapide)
```
Minimiser trajets longs!

### Après-midi (Retour)
```
Entrée → Salon → Cuisine → Chambres (relaxant)
```
Zone publique vers privée

### Soir (Repos)
```
Salon → Couloir → Chambre (isolé du reste)
```
Séparation nette zone privée importante

## Tailles Relatives Recommandées

Pour maison 100 m²:
- **Salon:** 25-35 m² (25-35%)
- **Cuisine:** 10-12 m² (10-12%)
- **Chambres:** 40-50 m² total (40-50%)
  - Chambre parent: 18-20 m²
  - Chambre enfant: 12-15 m² chacune
- **Salle de bain:** 6-8 m² (6-8%)
- **Circulation/Dégagement:** 8-12 m² (8-12%)

## Accessibilité (PMR - Personnes Mobilité Réduite)

Si accessibilité requise:
- Couloirs: minimum 1.5m (circulation fauteuil)
- Portes: minimum 0.9m (fauteuil passe 0.75m)
- Salle de bain: minimum 5-6m² pour maniabilité
- Chambre: minimum 10m² pour circulation
- Zéro escalier ou rampes d'accès
